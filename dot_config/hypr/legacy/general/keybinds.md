# ⌨️  HYPRLAND HELP
---------------------------------------------------------

### 🚀  SCRATCHPADS (Pyprland)
SUPER + Z              ::  Toggle Music Player (rmpc)
SUPER + SHIFT + Q      ::  Drop-down Terminal
SUPER + / (slash)      ::  Toggle Cheatsheet

### ⚙️  IMPORTANT SYSTEM BINDS
Fn    + F              ::  Toggle Power Profile (PPD)
SUPER + X              ::  Toggle Night Light (wlsunset)
SUPER + V              ::  Clipboard History (Cliphist)
Fn + SHIFT + F4/F5     ::  Brightness Up/Down Screenpad
Fn + SHIFT + 1/2/..    ::  Move windows between workspaces

### 🪟  IMPORTANT COMMANDS
s <name> <categories>  :: Search for packages on yay

### 📦  IMPORTANT THINGS
1) Made a udev rule for ppd : /etc/udev/rules.d/99-power-profiles.rules (manages performance profiles automatically)
2) Kooha for GIF recoding
3) Komikku for Manga reading
4) Use pypr reload
5) Use updatemirrors
6) Set Plymouth : 
    sudo plymouth-set-default-theme -l (to list installed themes)
    sudo plymouth-set-default-theme -R themename

---------------------------------------------------------
💡 Press 'q' to close this menu. Use arrow keys to scroll.
